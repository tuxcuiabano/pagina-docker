Bem-vindo ao Curso Docker!

O Docker é uma plataforma para construir, executar e gerenciar aplicativos em contêineres. Ele permite que os desenvolvedores empacotem o código e todas as suas dependências para que possam ser executados de forma rápida e confiável de um ambiente de computação para outro.

Neste curso, você aprenderá os fundamentos do Docker, incluindo como construir e executar contêineres, gerenciar imagens e implantar contêineres em produção.

O que você vai precisar

  • Um conhecimento básico de Linux
  • Um editor de texto
  • Uma instalação do Docker

Comece agora

Para começar este curso, siga estas etapas:

  1. Instale o Docker se ainda não o fez.
  2. Abra uma janela de terminal.
  3. Siga as instruções no próximo módulo para construir seu primeiro contêiner Docker.
